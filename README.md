# CSE471: System Analysis and Design

This course provides knowledge of systems analysis and design tools and techniques, including Data Flow Diagrams (DFD) and Entity Relationship (ER) diagrams. It involves analyzing system requirements and designing systems through project work using these methodologies. The course also includes a compulsory three hour laboratory session each week to provide practical experience and reinforce theoretical concepts.

## Course Objectives

1. Introduce Students with fundamental concepts of System analysis and design
2. Teach students about structural model of the system
3. Teach behavioral models of the system
4. Explain how to make a physical model of the system.
5. Teach students about managing programming processes and documentation
6. Help students to build a system using proper and modern tools
